# Changelog

All notable changes to the "Copilot Chat Extractor" extension will be documented in this file.

## [1.0.0] - 2025-01-19

### Added
- Initial release
- Export chat sessions to Markdown, JSON, or HTML
- Search across all Copilot chat history
- Sidebar tree view with workspace grouping
- Preview sessions in webview panel
- Bulk export all sessions
- Export recent N sessions
- Support for VS Code, VS Code Insiders, VSCodium, and Cursor
- Configuration options for default format, output directory, and detail level
- SQLite database extraction support
- Multiple chat format parsing (handles various Copilot versions)
